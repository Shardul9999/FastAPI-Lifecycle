Client Request
   ↓
Logging Middleware
   ↓
Auth Middleware
   ↓
Dependency Injection (get_db)
   ↓
Route Handler (/users)
   ↓
Response Returned
   ↓
Back through Middleware

1) Global exception Handler (HTTPexcpetion) : Handles all FastAPI HTTPException errors like ->
401 Unauthorized
404 Not Found
400 Bad Request

2) Generic Exception Handler : Handles all unexpected errors that are NOT HTTPException ->
ZeroDivisionError
ValueError
database crashes



Practical Rule You Can Remember
Place	                     Preferred Error Handling
Route	        ->             raise HTTPException(...)
Dependency	  ->             raise HTTPException(...)
Middleware	  ->             return Response(...)
